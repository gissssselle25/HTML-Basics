How do you use:

<ol>:

This tag defines an ordered list which can be numerical or alphabetical. The <li> tag is used to define each list item.

<ul>:

This tag defines an ordered list which is bulleted. You can use the <ul> tag and the <li> tag to create unordered lists.

<dl>:

This tag defines a description. The <dl> tag is used in conjunction with the <dd> tag. <dl> tag is used for defining terms/names and <dd> tag is used for describing each term/name.

Difference between text editor and word processor?

A text editor provides the ability to add, change, or remove characters from a file while a word processor has numbers of options which you can format the text or document.

What is the client-server model?

The client-server model is a model of interaction where a program sends a request to another program and awaits a response.

Provide a description and definition for the following network protocols:

FTP:

The file transfer protocol (FTP) is a process that involves the transfer of files between devices through a network. The process works when one party allows another to send or receive files through the internet.

HTTP:

The hypertext transfer protocol (HTTP) is an application-level protocol for distributed, collaborative, and hypermedia information systems. It is used to deliver data (HTML flies, image files, query results, etc.) on the World Wide Web.

TCP/IP:

The transmission control/internet protocol (TCP/IP) is a set of standardized rules that allow computers to communicate on a network such as the internet.
